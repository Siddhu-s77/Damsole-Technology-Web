# Damsole Technologies Website

A modern, responsive website for Damsole Technologies - an AI-powered digital agency focused on delivering innovative web solutions.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling navigation with animated dropdown menus
- **Service Showcase**: Display of various digital services offered
- **Portfolio Section**: Client testimonials and reviews
- **Contact Information**: Easy-to-find contact details and subscription form

## Technologies Used

- HTML5
- CSS3 (Custom styles with Bootstrap integration)
- JavaScript
- Bootstrap 5.3.0
- Font Awesome Icons
- Bootstrap Icons

## Sections

1. **Hero Section**: Eye-catching landing section with call-to-action
2. **About Us**: Company information and achievements
3. **Services**: Digital marketing, web design, UI/UX, and more
4. **Portfolio**: Client reviews and testimonials
5. **Pricing**: Transparent pricing packages
6. **Blogs**: Blog section (coming soon)
7. **Case Study**: Case studies (coming soon)
8. **Contact**: Contact form and company information

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Siddhu-ohale/Damsole-Technology-Webstie.git
```

2. Navigate to the project directory:
```bash
cd Damsole-Technology-Webstie
```

3. Open `index.html` in your web browser
