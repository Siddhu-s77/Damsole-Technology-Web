### Installation Steps

1. **Clone the repository:**
```bash
git clone https://github.com/Siddhu-ohale/Damsole-Technology-Webstie.git
```

2. **Navigate to the project directory:**

Edit CSS variables in `assets/css/style.css`:
```css

    --accent: #001E8E;  /* Primary blue */
    --text: #0f172a;    /* Text color */
```
